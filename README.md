# 🧩 拼豆图纸生成器

上传图片，自动生成拼豆珠子排列图纸。支持 MARD 品牌 289 色色卡，基于 CIE-Lab 色彩空间精准配色。

## ✨ 功能特性

- **图片转拼豆图纸**：上传任意图片，自动像素化并映射到拼豆色卡
- **CIE-Lab 精准配色**：感知均匀色彩空间，颜色匹配更准确
- **板子拼接**：支持多块板子自动拼接排列
- **10×10 定位线**：红色粗线划分 10×10 区域，方便定位
- **颜色容差**：合并相近颜色，忽略羽化边缘噪点
- **最大颜色数**：限制用色种类，简化图纸
- **手动编辑**：逐格改色、批量替换颜色
- **滚轮缩放**：鼠标滚轮自由缩放查看
- **导出 PNG**：一键导出完整图纸图片

## 🚀 快速开始

### 环境要求

- Python 3.8+
- pip

### 安装运行

```bash
# 克隆项目
git clone https://github.com/Clyde323/pixel-beads-generator.git
cd pixel-beads-generator

# 安装依赖
pip install -r requirements.txt

# 启动服务
python app.py
```

打开浏览器访问 **http://localhost:5000**

## 📖 使用说明

### 基本流程

1. **上传图片**：点击或拖拽上传 JPG/PNG/BMP 图片
2. **设置板子规格**：默认 29×29（标准拼豆板）
3. **调整精密程度**：滑块控制分辨率，自动计算板子数量
4. **颜色优化**：设置容差值合并相近颜色
5. **生成图纸**：点击生成按钮
6. **手动编辑**（可选）：开启编辑模式，逐格改色或批量替换
7. **导出**：下载 PNG 格式图纸

### 参数说明

| 参数 | 说明 | 默认值 |
|------|------|--------|
| 板子规格 | 单块板子的格数（长×宽） | 29×29 |
| 精密程度 | 总分辨率，自动计算板子数量 | 29 |
| 颜色容差 | ΔE* 值，合并相近颜色 | 2 |
| 最大颜色数 | 限制用色种类 | 不限 |

### 编辑模式

- **画笔色号**：选择色号后点击格子修改
- **批量替换**：选择原色→目标色，一键替换全部

## 🎨 色卡数据

MARD 拼豆色卡数据来源：[pixel-beads.com](https://www.pixel-beads.com/zh/mard-bead-color-chart)

包含 **289 色**，覆盖 A~ZG 全系列色号。

## 🛠️ 技术栈

- **后端**：Python + Flask + Pillow + NumPy
- **前端**：原生 HTML/CSS/JavaScript + Canvas
- **颜色算法**：CIE-Lab 色彩空间（RGB → XYZ → Lab）
